# Zebra ![Zig](https://img.shields.io/badge/Zig-%23F7A41D.svg?style=for-the-badge&logo=zig&logoColor=white)

A zero dependency all-in-one config loader for Zig. Supports reading dotenv, toml,
yaml and os env.
